In this tutorials section we introduce the handling of the three core classes which form the foundation of SSMTool.

  • Setting up a Dynamical System object
  • Setting up a Manifold object
  • Setting up a SSM object

Each of these classes is responsible for a certain part of the analysis, as displayed in the figure below.

After having properly understood these core routines, the methods provided for the analysis of dynamical systems, which do build on these fundamental classes, can be understood. The essential functionalities offered encompass the computation of

  • Backbone Curves
  • Forced Response Curves
  • Continuation of the Reduced Dynamics
  • Stability Diagrams

Examples

In the Examples section, a wide variety of examples that employ these routines are provided. The library of examples encompasses small academic examples that exhibit interesting physical phenomena as well as large, finite-element examples - all of which are analysed using the reduced dynamics on invariant SSMs.
