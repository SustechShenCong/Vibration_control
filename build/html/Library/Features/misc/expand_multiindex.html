function S = expand_multiindex(M,P)

EXPAND_MULTIINDEX

This function returns the multi-index expansion for the multi-index pair over the column vectors in the matrix , where the coefficients and indices are fields of the data structure M.

if isempty(M.ind)
    S = 0;
else
    nt = size(P,2);
    C = M.coeffs;
    I = M.ind;
    n_I = size(I,1);
    % evaluate monomials
    P_I = prod(kron(P.',ones(n_I,1)) .^ repmat(I,nt,1),2);
    % evaluate polynomials
    S = C*reshape(P_I,[],nt);
end
        
end
